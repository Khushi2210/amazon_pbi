# Amazon Prime Content Insights Dashboard

## Project Overview
With increasing competition in the streaming industry, understanding content distribution, ratings, and audience trends is critical.  
This Power BI project analyzes Amazon Prime’s library of movies and television shows to extract meaningful insights related to content composition, ratings, genres, geographic distribution, and historical trends.

---

## Project Objectives
The objective of this project is to develop an interactive Power BI dashboard that enables stakeholders to explore Amazon Prime’s content library from multiple analytical perspectives, including:

- Distribution between movies and television shows
- Most common content rating categories
- Growth in content volume over time
- Dominant genres and countries of production
- Identification of frequently contributing directors

---

## Dataset Description
The dataset contains the following attributes:

| Column Name | Description |
|------------|------------|
| show_id | Unique identifier for each title |
| type | Content type (Movie or TV Show) |
| title | Title of the content |
| director | Director name(s) |
| cast | Primary cast members |
| country | Country of production |
| date_added | Date added to Amazon Prime |
| release_year | Year of release |
| rating | Age rating classification |
| duration | Duration or number of seasons |
| listed_in | Genre or category |
| description | Brief content summary |

---

## Key Insights
The dashboard provides the following aggregated insights:

- Total Titles Analyzed: 9,655  
- Distinct Rating Categories: 25  
- Unique Genres Identified: 519  
- Unique Directors: 5,771  
- Content Timeline Covered: 1920 to 2021
